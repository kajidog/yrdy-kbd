# Viewer

React app for finding lives, watching live screens over KVS WebRTC as a
`VIEWER`, and playing recordings over HLS.

## Structure

```
src/
  app/          App entry component + global styles
  features/     home (search/catalog) and watch (live/recording player) screens
  components/   HlsPlayer
  graphql/      GraphQL operations (typed via codegen's graphql())
  gql/          generated by GraphQL Code Generator — do not edit
  lib/          KVS WebRTC viewer runtime, client id
```

## Run

From the repository root (npm workspaces):

```sh
npm install
npm run dev -w @yrdy-kbd/viewer -- --port 5174
```

Set `VITE_BFF_BASE_URL` if the BFF is not running on `http://localhost:8080`.

After changing `packages/schema/schema.graphql` or any GraphQL operation:

```sh
npm run codegen -w @yrdy-kbd/viewer
```

## Notes

- The publisher-generated watch link includes `liveId`.
- Each browser tab generates its own KVS viewer `clientId`.
- AWS credentials are never configured in this app.
