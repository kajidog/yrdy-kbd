# Publisher

React app for creating lives and broadcasting a screen share over KVS WebRTC
as the `MASTER`.

## Structure

```
src/
  app/          App entry component + global styles
  features/     dashboard (state owner), lives (create/list panels), broadcast (preview panel)
  graphql/      GraphQL operations (typed via codegen's graphql())
  gql/          generated by GraphQL Code Generator — do not edit
  lib/          KVS WebRTC publisher runtime
```

## Run

From the repository root (npm workspaces):

```sh
npm install
npm run dev -w @yrdy-kbd/publisher -- --port 5173
```

Set `VITE_BFF_BASE_URL` if the BFF is not running on `http://localhost:8080`.

After changing `packages/schema/schema.graphql` or any GraphQL operation:

```sh
npm run codegen -w @yrdy-kbd/publisher
```

## Notes

- Uses `navigator.mediaDevices.getDisplayMedia({ video: true, audio: true })`.
- AWS credentials are never configured in this app.
- KVS signaling WSS URLs are signed by the Go BFF through the SDK `requestSigner` hook.
